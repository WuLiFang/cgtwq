## 安装

因为使用了 PEP508 URL 依赖, 需求 pip 版本高于 18.1

```shell
pip install https://github.com/WuLiFang/cgtwq/archive/v3.0.0-beta.0.zip#egg=cgtwq
```

## 开发

使用 [GNU Make](https://www.gnu.org/software/make/)。

```shell
# 安装依赖
make
# 构建文档
make docs
# 运行测试
make test
```
